## FastCraft Frontend (React + Vite)

Фронтенд для сайта FastCraft: главная страница, магазин доната (категории, карточки, модалка покупки), страница правил и UI‑компоненты в фиолетовой теме.

---

## Что внутри

- **Главная**: интро + статистика сервера
- **Кнопка “Копировать”**: копирует `FastCraft.me:19132` и меняет текст на “Скопировано!”
- **Магазин**:
  - категории: привилегии / кейсы / FastCoins / услуги
  - модалка покупки (никнейм, промокод, чекбокс согласия)
  - блок “Последние покупки” (моки)
- **Правила**:
  - выбор режима (Гриф / Анархия)
  - отображение правил в виде аккордиона
- **Модалки**:
  - покупка
  - “Как начать играть” (шаги + одна кнопка)

---

## Стек

- React + Vite
- CSS Modules
- React Router

---

## Установка и запуск

В папке `frontend`:

```bash
npm install
npm run dev
```

Обычно проект открывается на `http://localhost:5173/`.

---

## Сборка

```bash
npm run build
npm run preview
```

---

## Структура (кратко)

- `src/pages/homepage/components/` — блоки главной страницы (интро, статистика, about, магазин)
- `src/pages/rulesPage/components/` — страница правил + аккордионы
- `src/components/ui/` — UI (модалки)
- `src/assets/` — картинки (heads/case и т.д.)
